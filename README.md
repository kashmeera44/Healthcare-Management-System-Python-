# Healthcare Management System

## Overview

The Healthcare Management System is a Python-based application developed using Object-Oriented Programming (OOP) concepts. This project helps manage patient records and hospital operations through a simple menu-driven interface.

## Features

* View Patient Details
* Admit New Patients
* Discharge Patients
* Exit System
* Patient Record Management
* User-Friendly Interface

## Technologies Used

* Python
* Object-Oriented Programming (OOP)
* Classes and Objects
* Functions
* Conditional Statements
* Loops

## How It Works

1. Admit a patient into the system.
2. View patient details and records.
3. Discharge patients when treatment is completed.
4. Exit the application safely.
